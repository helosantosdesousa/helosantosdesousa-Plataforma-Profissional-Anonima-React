# Plataforma Profissional Anônima
 ## Visão geral
A Plataforma Profissional Anônima é uma aplicação desenvolvida em React (TypeScript) cujo objetivo é permitir que profissionais se cadastrem e interajam de forma anônima para compartilhar experiências, fazer publicações, encontrar outros profissionais etc.

## Tecnologias utilizadas
* React com Typscript

## Funcionalidades Principais (protótipo)
* Login com perfil de teste
* Páginas de perfil de usuário
* Visualização de conteúdos
* Navegação entre páginas
* Interface de DM
* Vagas de emprego com API the muse
* Interface de Mtachmaking

# Instalação & Setup
``` # clonar o repositório
git clone https://github.com/helosantosdesousa/helosantosdesousa-Plataforma-Profissional-Anonima-React.git

# entrar na pasta do projeto
cd helosantosdesousa-Plataforma-Profissional-Anonima-React

# instalar dependências
npm install


# rodar em modo de desenvolvimento
npm start

```
