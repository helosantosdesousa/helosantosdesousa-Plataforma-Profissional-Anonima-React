# Plataforma Profissional Anônima

## Visão geral

A **Plataforma Profissional Anônima** é uma aplicação desenvolvida em **React Native com TypeScript (Expo)** cujo objetivo é permitir que profissionais se cadastrem e interajam de forma anônima para compartilhar experiências, fazer publicações, encontrar outros profissionais etc.

## Tecnologias utilizadas

* React Native (Expo)
* TypeScript
* EAS Build

---

## Instalação & Setup

```bash
# clonar o repositório
git clone https://github.com/helosantosdesousa/helosantosdesousa-Plataforma-Profissional-Anonima-React.git

# entrar na pasta do projeto
cd helosantosdesousa-Plataforma-Profissional-Anonima-React

# instalar dependências
npm install
```

### Rodando o projeto

**Emulador Android (recomendado):**

```bash
# iniciar o projeto
npx expo start

# escolha a opção "a" para abrir no Android Emulator
```

**No navegador (Web):**

```bash
# iniciar o projeto
npx expo start

# escolha a opção "w" para abrir no navegador
```
